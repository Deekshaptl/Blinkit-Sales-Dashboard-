1️⃣ Project Title
# Blinkit Sales Dashboard 📊

2️⃣ Project Description (Overview)
## Project Overview
The Blinkit Sales Dashboard is a data analytics project built using Power BI. 
The goal of this project is to analyze sales performance, customer preferences, 
and outlet performance of Blinkit.

The dashboard helps visualize important business metrics such as total sales, 
average sales, item categories, outlet size, and customer ratings.

3️⃣ Tools Used
## Tools and Technologies

- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization

4️⃣ Dataset Information
## Dataset Information
The dataset includes the following columns:

- Item Identifier
- Item Type
- Item Fat Content
- Item Weight
- Outlet Size
- Outlet Location
- Outlet Establishment Year
- Item Visibility
- Sales
- Rating

5️⃣ Dashboard KPIs
## Key Performance Indicators (KPIs)

The dashboard displays the following metrics:

- Total Sales: $1.20M
- Average Sales: $141
- Total Number of Items: 8523
- Average Rating: 3.9

6️⃣ Dashboard Visualizations
Explain the charts used in the dashboard.
## Dashboard Visualizations

The dashboard contains multiple charts such as:

1. Total Sales by Fat Content
2. Sales by Item Type
3. Sales by Outlet Size
4. Sales by Outlet Location
5. Outlet Establishment Trend

7️⃣ Filters Used
Explain the filter panel in your dashboard.
## Filters
Users can interact with the dashboard using the following filters:

- Outlet Location
- Outlet Size
- Item Type

8️⃣ Key Insights
Explain what insights you discovered.
## Key Insights

- Regular fat products generate higher sales compared to low-fat items.
- Tier 1 cities contribute the highest sales.
- Medium and large outlets generate more revenue.
- Fruits and snacks are among the top-selling categories.

9️⃣ Project Outcome
Explain the purpose of the dashboard.
## Project Outcome
This dashboard helps businesses understand sales performance, 
product demand, and outlet performance to make better business decisions.
