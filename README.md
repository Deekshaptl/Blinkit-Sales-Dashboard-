# Blinkit Sales Dashboard 📊

## Project Overview
This project analyzes sales performance of Blinkit using a Power BI dashboard. 
The goal of the project is to understand sales trends, customer preferences, 
and outlet performance using data visualization.

## Tools Used
- Power BI
- Excel Dataset
- Data Cleaning
- Data Visualization

## Dataset Information
The dataset contains information about:
- Item Type
- Outlet Size
- Outlet Location
- Fat Content
- Sales Amount
- Rating
- Establishment Year

## Dashboard KPIs
The dashboard shows the following key performance indicators:

- Total Sales: $1.20M
- Average Sales: $141
- Number of Items: 8523
- Average Rating: 3.9

## Dashboard Visualizations
The dashboard includes the following charts:

1. Total Sales by Fat Content
2. Sales by Item Type
3. Sales by Outlet Size
4. Sales by Outlet Location
5. Outlet Establishment Year Trend

## Filters
The dashboard allows filtering data by:
- Outlet Location
- Outlet Size
- Item Type

## Key Insights
- Regular fat products generate higher sales.
- Tier 1 cities generate the highest revenue.
- Medium and large outlets contribute most to total sales.
- Fruits and snack items are top-selling categories.

## Project Outcome
This dashboard helps businesses understand:
- Customer purchasing patterns
- Sales performance by outlet
- Product demand trends
